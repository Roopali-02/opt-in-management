# opt-in-management
A user-friendly interface to manage opt-in and opt-out responses. Built with HTML, CSS,JS and Bootstrap 5.

## Setup
- Clone the repo.
- Open `index.html` in a browser to view the project.

## Technologies Used
- HTML
- CSS
- Bootstrap 5 (via CDN)
- Font Awesome (via CDN)
- Google Fonts (via CDN)
